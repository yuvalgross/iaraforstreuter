## Step 1: Push Code to GitHub

```bash
cd vercel-portfolio
git init
git add .
git commit -m "Iara portfolio with periwinkle design"
git remote add origin https://github.com/YOUR_USERNAME/iaraforstreuter.git
git push -u origin main
```

## Step 2: Vercel Auto-Deploys

Once pushed to GitHub, Vercel automatically:
- ✅ Detects your changes
- ✅ Builds (reads vercel.json)
- ✅ Deploys to iaraforstreuter.vercel.app
- ✅ Shows deployment status in your project dashboard

**Check deployment:** https://vercel.com/yuval-s-projects6/iaraforstreuter
